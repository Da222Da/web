流程控制 ​

在 JavaScript 中，流程控制语句用于控制程序的执行顺序。它们允许程序在不同的条件下执行不同的代码块，或者重复执行某段代码直到满足特定条件为止。

JavaScript 中的流程控制语句主要有三种：分支语句、循环语句和跳转语句。其中，分支语句用于根据条件执行不同的代码块；循环语句用于重复执行某段代码直到满足特定条件为止；跳转语句则用于在程序中直接改变程序的执行顺序。

js
// If 条件判断
if (a > b) {
	console.log("a大于b");
} else if (a < b) {
	console.log("a小于b");
} else {
	console.log("a等于b");
}

// Switch 分支语句
switch (fruit) {
	case "apple":
		console.log("苹果");
		break;
	case "banana":
		console.log("香蕉");
		break;
	case "orange":
		console.log("橙子");
		break;
	default:
		console.log("未知水果");
}
